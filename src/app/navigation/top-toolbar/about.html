About ElderDrug

Elderdrug.com was created to make the AGS Beers Criteria easier to search. It uses RxNORM for drug data, Firebase for hosting, and Angular for front-end design. Many drugs in this database have been determined by querying Class/Therapeutic Category members (e.g. Cyclooxygenase Inhibitors), and using the resulting generic RxCUIs to query brand names. This generates thousands of drugs, so I cannot guarantee accuracy for all entries. However, I will continue to test and fix any issues as they arise.
